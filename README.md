# Worlds Edge Wellness - Therapist Website
# By: Drew Ratliff

A professional, responsive website for Megan McCraw, a therapist serving Hendersonville, North Carolina. This website features a calm and approachable design optimized for both web and mobile devices.

## Project Overview

**Therapist:** Megan McCraw  
**Location:** Hendersonville, NC  
**Website Type:** Professional therapy practice website  
**Design Theme:** Calm and approachable  
**Target Audience:** Potential therapy clients in the Hendersonville area

## Website Structure

The website consists of four main pages:

### 1. Home Page (`index.html`)
- Welcome message and introduction
- Overview of services
- Call-to-action for scheduling
- Contact information
- Professional headshot and brief bio

### 2. About Me
- Detailed therapist biography
- Professional credentials and certifications
- Therapeutic approach and philosophy
- Personal background and experience
- Professional photos

### 3. Offerings
- Detailed service descriptions
- Therapy types offered (individual, couples, family, etc.)
- Specializations and areas of expertise
- Session formats (in-person, telehealth)
- Pricing information (if applicable)

### 4. FAQ
- Common questions about therapy process
- Insurance and payment information
- Scheduling and cancellation policies
- Confidentiality and privacy information
- What to expect in first session

## Technical Specifications

### Responsive Design
- **Mobile-first approach** for optimal mobile experience
- **Breakpoints:**
  - Mobile: 320px - 768px
  - Tablet: 768px - 1024px
  - Desktop: 1024px+
- **Touch-friendly navigation** for mobile devices
- **Optimized images** for different screen sizes

### Design Theme: Natural Mountain Retreat
- **Color Palette (Inspired by worldsEdgeShack.JPG):**
  - Primary: Forest Green (#4A5D23, #6B7B3A)
  - Secondary: Warm Cream (#F5F1E8, #FEFCF7)
  - Accent: Rustic Brown (#8B4513, #A0522D)
  - Text: Charcoal (#2F2F2F, #4A4A4A, #6B6B6B)
- **Hero Background:**
  - worldsEdgeShack.JPG - Authentic mountain cabin at sunset/sunrise
  - Natural colors with subtle dark overlay for text readability
- **Typography:**
  - Headings: Nunito (Google Fonts)
  - Body Text: Source Sans Pro (Google Fonts)
  - Appropriate font sizes for accessibility
  - Excellent contrast ratios with charcoal text
- **Visual Elements:**
  - Soft, rounded corners
  - Gentle shadows and subtle animations
  - Nature-inspired imagery
  - Plenty of white space

### Performance Optimization
- Optimized images (WebP format where supported)
- Minified CSS and JavaScript
- Fast loading times
- SEO-friendly structure

## Development Setup

### Prerequisites
- Modern web browser
- Text editor or IDE
- Local web server (optional for development)

### Getting Started
1. Clone or download the project files
2. Open `index.html` in your web browser
3. For development with live reload, use a local server:
   ```bash
   # Using Python 3
   python -m http.server 8000
   
   # Using Node.js (with live-server)
   npx live-server
   ```
